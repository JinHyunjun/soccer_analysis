# 선수 경기 지표 한국어 사전

아래 정의는 팬에게 보여줄 공통 설명의 초안이다. 실제 표시 문구는 계약한 공급자의 정의와 일치하도록 `metric_definition_version`별로 관리해야 한다. 특히 도움, 태클, 경합, 키패스, xA는 공급자별 판정 기준이 다를 수 있다.

## 기본 정보와 평점

| 지표 | 뜻 | 해석할 때 주의할 점 |
|---|---|---|
| 출전 시간 | 선수가 실제로 뛴 분 | 추가시간 처리 방식 때문에 90을 넘거나 공급자 간 차이가 날 수 있음 |
| 선발/교체 | 선발 출전, 교체 투입·아웃 여부 | 벤치에만 있던 선수는 출전 통계와 분리 |
| 포지션 | 해당 경기에서 맡은 역할 | 등록 포지션과 실제 경기 포지션이 다를 수 있음 |
| 평점 | 여러 경기 이벤트를 가중 합산한 공급자 고유 점수 | FotMob·Sofascore·WhoScored 점수는 산식과 척도가 달라 서로 직접 비교하면 안 됨 |
| 90분당 | `값 / 출전분 × 90` | 짧게 뛴 선수는 값이 과장될 수 있어 최소 출전분 필터 필요 |

## 득점과 슈팅

| 지표 | 뜻 | 해석할 때 주의할 점 |
|---|---|---|
| 골 | 해당 선수에게 기록된 득점 | 자책골과 승부차기 득점은 일반 골과 분리 |
| 도움 | 득점 직전 팀 동료의 마지막 유효 기여 | 대회·공급자별로 굴절, 리바운드, PK 획득의 인정 기준이 다름 |
| 슈팅 | 득점을 의도한 시도 | 유효, 빗나감, 블록으로 나눔 |
| 유효 슈팅 | 골이 되거나 골키퍼/마지막 수비수가 막지 않았다면 골문 안으로 들어갈 슛 | 골대를 맞고 나오면 보통 빗나간 슛이며 공급자 정의 확인 필요 |
| 빗나간 슈팅 | 골문을 벗어난 슈팅 | 수비에게 막힌 슈팅과 구분 |
| 블록된 슈팅 | 골문으로 향하는 과정에서 필드 플레이어에게 막힌 슈팅 | 골라인의 마지막 수비 블록은 유효 슈팅으로 잡힐 수 있음 |
| 골대 강타 | 골포스트나 크로스바를 맞고 골이 되지 않은 시도 | 한 슛이 여러 번 골대를 맞아도 한 번으로 집계될 수 있음 |
| 빅찬스 | 일반적으로 득점이 매우 기대되는 1대1·근거리 기회 | 사람의 판정이 들어가는 공급자 고유 지표이며 페널티는 대개 빅찬스 |
| 빅찬스 미스 | 빅찬스에서 득점하지 못했거나 슈팅 자체를 하지 못한 경우 | 단순히 높은 xG 슛의 실패와 완전히 같은 개념은 아님 |
| xG | 슈팅 당시 위치·각도·상황이 골이 될 확률을 0~1로 추정한 값 | 선수의 의도나 결과가 아니라 기회의 질. 모델마다 값이 다름 |
| npxG | 페널티 슈팅을 제외한 xG | 페널티 기회 차이를 제거해 오픈플레이 위협을 비교하기 좋음 |
| xGOT/PSxG | 유효 슈팅이 실제 골문 어디로 향했는지까지 반영한 사후 기대득점 | 슈팅 전에 계산하는 xG와 달리 마무리 위치를 반영; 골키퍼 선방 평가에 유용 |
| 슈팅 정확도 | 유효 슈팅 ÷ 전체 슈팅 | 일반적으로 블록 슈팅·자책골 제외 여부를 확인 |
| 슈팅 전환율 | 득점 ÷ 전체 슈팅 | 짧은 기간에는 운과 표본 변동이 큼 |

## 패스와 창의성

| 지표 | 뜻 | 해석할 때 주의할 점 |
|---|---|---|
| 패스 시도/성공 | 동료에게 공을 전달하려 한 횟수와 성공 횟수 | 크로스, 스로인, GK 던지기를 패스에 포함하는지가 다를 수 있음 |
| 패스 성공률 | 성공 패스 ÷ 시도 패스 | 안전한 횡패스가 많은 선수에게 유리하므로 난도·진행성도 같이 봐야 함 |
| 키패스 | 동료의 슈팅으로 바로 이어진 마지막 패스 중 골이 되지 않은 것 | 도움과 합치면 보통 `찬스 창출`이 됨 |
| 찬스 창출 | 도움 + 키패스 | 공급자가 이 정의를 쓰는지 확인 필요 |
| 빅찬스 창출 | 동료에게 빅찬스를 만든 패스 | 실제 도움 여부와 별개 |
| xA | 완료된 패스가 도움으로 이어질 가능성을 추정한 값 | 일부 모델은 이어진 슈팅의 xG를 사용하고 일부는 패스 자체 모델을 사용 |
| 롱패스 | 일정 거리 이상 길게 보낸 패스 | 거리 기준이 공급자마다 다름 |
| 스루패스 | 수비 라인을 갈라 동료가 달려가 받을 수 있게 보낸 패스 | 의도 판정이 포함됨 |
| 크로스 | 측면에서 골문 앞 중앙 지역의 동료를 겨냥한 전달 | 단순한 측면 패스와 구분 |
| 전진 패스 | 공격 방향으로 의미 있게 전진한 패스 | Opta 예시는 공격 2/3에서 골까지 거리를 25% 이상 줄인 완료 패스; 공급자별 정의가 큼 |
| 파이널 서드 패스 | 상대 진영 마지막 1/3 구역으로 들어가거나 그 안에서 한 패스 | 시작/도착 기준을 화면에 명시 |
| 페널티 박스 진입 패스 | 상대 박스 안으로 성공시킨 패스 | 크로스 포함 여부를 확인 |

## 점유, 드리블, 운반

| 지표 | 뜻 | 해석할 때 주의할 점 |
|---|---|---|
| 터치 | 선수가 공에 닿은 것으로 기록된 이벤트의 합 | 이벤트 데이터 기반이라 모든 미세한 접촉을 뜻하지는 않음 |
| 상대 박스 안 터치 | 상대 페널티 구역에서의 터치 | 공격 위치 점유를 보여주지만 기회의 질 자체는 아님 |
| 드리블 시도 | 공을 가진 채 상대 한 명을 제치려 한 시도 | 단순 운반과 다름 |
| 드리블 성공 | 상대를 제치고 소유를 유지한 드리블 | 수비수를 완전히 지나야 하는지 등 판정 차이가 있음 |
| 볼 빼앗김 | 소유 중 상대에게 직접 공을 잃음 | 패스 실패나 나쁜 터치로 잃은 공과 분리될 수 있음 |
| 소유권 상실 | 패스 실패, 나쁜 터치, 드리블 실패 등으로 팀 소유가 끝난 횟수 | 공급자가 공개한 합산 구성요소 확인 필요 |
| 볼 운반 | 공을 통제하며 일정 거리 이상 이동 | Opta 정의 예시는 5m 이상 이동 |
| 전진 운반 | 공격 방향으로 의미 있게 공을 이동한 운반 | 거리·방향 기준이 공급자마다 다름 |

## 수비와 경합

| 지표 | 뜻 | 해석할 때 주의할 점 |
|---|---|---|
| 태클 | 공을 가진 상대에게 지상 도전을 해 공에 접촉한 수비 행동 | 태클 성공/승리는 이후 소유권까지 확보했는지에 따라 달라질 수 있음 |
| 가로채기 | 상대 패스 경로를 읽고 먼저 차단해 공을 얻은 행동 | 단순 패스 블록과 구분 |
| 클리어링 | 특정 동료를 노리지 않고 위험 지역에서 공을 걷어낸 행동 | 수비 압박을 많이 받은 팀의 선수에게 숫자가 커질 수 있음 |
| 슛 블록 | 필드 플레이어가 상대의 골문 방향 슈팅을 막음 | 원래 빗나갈 슛은 블록으로 기록하지 않을 수 있음 |
| 볼 회복 | 어느 팀도 소유하지 않던 공이나 상대가 직접 흘린 공을 확보 | 가로채기·태클과 겹치지 않는 별도 이벤트일 수 있음 |
| 드리블 허용 | 상대 드리블러에게 제쳐진 횟수 | 수비 역할과 대면 횟수의 영향을 크게 받음 |
| 지상 경합 | 지면에서 벌어진 상대와의 직접 경쟁 | 태클·드리블·파울 등이 포함되는 방식은 공급자별 차이가 큼 |
| 공중 경합 | 두 선수가 공중볼을 놓고 경쟁한 상황 | 공을 팀이 유지했는지와 개인이 첫 접촉했는지는 다를 수 있음 |
| 경합 승률 | 이긴 경합 ÷ 전체 경합 | 작은 표본과 역할 차이에 주의 |
| 실점 유발 실수 | 선수의 명확한 실수가 직접 상대 골로 연결 | 주관 판정이며 실수로 슈팅만 내준 경우와 분리 |
| 슈팅 유발 실수 | 선수 실수가 직접 상대 슈팅으로 연결 | 골이 되면 실점 유발 실수와 중복 여부 확인 |
| 상대 진영 3선 회수 | 상대 골문에 가까운 마지막 1/3에서 소유권을 되찾음 | 전방 압박의 결과를 간단히 보여줌 |
| 압박 | 공을 가진 상대가 판단하기 어렵게 가까이 접근한 수비 행동 | 대부분의 저가 집계 API에는 안정적으로 제공되지 않음 |

## 규율과 세트피스

| 지표 | 뜻 | 해석할 때 주의할 점 |
|---|---|---|
| 파울 범함 | 심판이 반칙으로 선언해 프리킥·PK가 주어진 행위 | 어드밴티지 후 카드만 나온 경우 집계 규칙 확인 |
| 피파울 | 상대 반칙으로 팀이 프리킥·PK를 얻은 행위 | 핸드볼 등은 특정 선수의 피파울로 주지 않을 수 있음 |
| 오프사이드 | 플레이에 관여해 오프사이드가 선언된 횟수 | 오프사이드 위치에 있었던 모든 선수를 뜻하지 않음 |
| 경고/퇴장 | 옐로카드, 두 번째 경고 퇴장, 직접 퇴장 | 두 번째 경고와 직접 퇴장을 분리 저장 |
| PK 획득/허용 | 반칙 등으로 페널티킥을 얻거나 내준 선수 | 팀 단위 사건과 선수 귀속을 분리 |
| PK 성공/실패 | 페널티킥 득점 또는 실패 | 승부차기와 경기 중 PK를 분리 |

| 지표 | 뜻 | 해석할 때 주의할 점 |
|---|---|---|

| 클린시트 | 일정 출전 조건을 충족하며 무실점으로 끝낸 경기 | 교체 선수 인정 기준이 대회/서비스마다 다름 |

| PK 선방 | 경기 중 페널티킥을 막은 횟수 | 골대를 맞거나 빗나간 PK는 보통 선방 아님 |
| 공중볼 처리 | 크로스 등을 잡거나 쳐낸 행동 | 캐치, 펀칭, 하이클레임을 분리할 수 있음 |
| 스위퍼 행동 | 박스 밖으로 나와 위험을 제거한 GK 수비 행동 | 위치 이벤트가 필요 |

## 고급·시각 지표

| 지표 | 뜻 | 사용 제안 |
|---|---|---|
| 히트맵 | 기록된 터치/행동 위치의 밀도 | 활동량이 아니라 공과 관련된 활동 위치로 설명 |
| 평균 위치 | 기록 이벤트 좌표의 평균 | 실제 전술 위치를 완전히 대표하지 않음 |
| 슈팅 맵 | 각 슛의 위치, 결과, xG를 피치 위에 표시 | 가장 먼저 구현할 고급 시각화 |
| 패스 맵 | 시작·도착점과 성공 여부 | 선수 단위는 선의 수가 많으므로 필터 필요 |
| 수비 맵 | 태클·가로채기·클리어링·회복 위치 | 행동 유형별 색과 툴팁 제공 |
| xGChain | 슈팅으로 끝난 점유에 관여한 선수에게 해당 슈팅 xG를 연결 | 공급자 고유 정의로 표시 |
| xGBuildup | xGChain에서 슈터와 마지막 찬스 제공자를 제외한 빌드업 기여 | 수비수·중앙 미드필더의 간접 기여 탐색에 유용 |
| OBV | 각 온볼 행동이 팀의 득점/실점 가능성을 얼마나 바꿨는지 평가 | StatsBomb 독점 모델이므로 계약 없이 명칭·값 복제 금지 |
| PPDA | 특정 지역에서 허용한 상대 패스 수 ÷ 그 지역의 수비 행동 수 | 주로 팀 압박 지표이며 낮을수록 강한 압박으로 해석 |

## 팬에게 함께 보여줄 해석 규칙

- 숫자 하나로 “잘했다/못했다”를 단정하지 않는다.
- 중앙 수비수의 패스 성공률과 공격수의 패스 성공률은 역할이 다르다.
- 합계와 90분당을 함께 제공하고 최소 출전 시간을 표시한다.
- xG와 xA는 기대값이지 실제 득점·도움의 판정 변경값이 아니다.
- 평점에는 공급자 이름을 붙인다. 예: `Sportmonks 평점 7.4`.
- 다른 공급자의 평점·xG는 한 리더보드에서 혼합하지 않는다.
